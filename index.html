TP BOUTIQUE (AJAX – JSON) passage à PHP
A partir des fichiers fournis, vous êtes chargé de :

  1. Créer un fichier produits.php qui renvoie les données JSON
  2. Dans le JS intégrer un nouveau fetch qui charge ce fichier à chaque recherche
  3. Créer et peupler la base BD correspondant au Json
  4. Modifier produits.php pour requêter les données sur la BD et les renvoyer en Json
  5. Créer et peupler la base BD correspondant au Json
  6. Dans le JS modifier le nouveau fetch pour qu’il envoie les choix du formulaire client au fichier produits.php
  7. Modifier produits.php pour qu’il ne renvoie que les données qui correspondent aux choix du client.
